How to use it

Options

Options can be specified in three ways:

  • On the command line
  • In a config file
  • On the standard input
C++ Programmers: The options are handled by the Boost library Program Options. Please refer to Boost's documentation for more information.

Build information

The build process is managed by Cmake. The easiest way to build is to open in your terminal the root directory of the fishFlow repository and to type: 

mkdir build
cd build
cmake ..
make
Tips: You can use different generators. For instance to generate a Xcode project, use cmake -G Xcode ... You can also use the ncurses interface to cmake with ccmake ...
